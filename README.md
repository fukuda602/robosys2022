## sisokuコマンド
* 1 ～ N番目までの範囲内の四則演算を行うコマンドです。少数にも対応しています。

## インストール
```
git clone https://github.com/fukuda602/robosys2022
```
## 必要ソフトウェア
* python 3.7 ～ 3.10

## テスト環境
* ubuntu20.04
