## sisokuコマンド
* N ～ M番目までの範囲内の四則演算を行うコマンドです。少数にも対応しています。
 * 出力は加法、減法、乗法、除法の順です。

## 実行方法
* N,Mには適当な数字を入れてください。
```
seq N M | ./sisoku
```
## インストール
```
git clone https://github.com/fukuda602/robosys2022
```
## 必要ソフトウェア
* python 3.7 ～ 3.10

## テスト環境
* ubuntu20.04
